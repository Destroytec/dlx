A DLX interpreter with following features:

  1. Editor to write code
  2. An overview over the registry and memory entries
  3. Save and load feature with autosave and example programs
  4. (Hopefully) helpful error messages for debugging

How to use:

  1. While you write your program, you can change between interpreter mode and editor mode.
  2. The editor mode is the normal write mode, where you can write your program.
  3. The interpreter mode locks the editor (but not your memory/registry entries) and you can execute your program complete or in steps.
